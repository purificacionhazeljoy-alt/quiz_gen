**QuizLab - Quiz Generator System**

**Overview**
QuizLab is a web-based Quiz Generator and Management System designed for teachers and students.  
The system provides real-time quiz monitoring, user management, cheating detection, and instant result tracking.

**Objectives**
•	To provide an easy-to-use quiz creation platform.
•	To organize quiz questions and assessments.
•	To help users generate quizzes quickly and efficiently.
•	To improve the assessment and learning experience.

**Features**
Admin
-  Manage users
-  View system activity logs
-  Monitor cheating logs
-  Manage reports and analytics

Teacher
-  Create and manage quizzes
-  Add and edit questions
-  View student submissions
-  Monitor real-time quiz activity

Student
-  Take quizzes online
-  View scores instantly
-  Real-time quiz experience


**System Features**
-  Real-time updates using Pusher
-  Anti-cheating activity detection
-  Secure authentication system
-  Role-based access control
-  Responsive dashboard UI
-  Quiz timer support
-  Activity logging system



**How to Use the System**

Step 1: Login
Open the system and login using your account credentials.

Step 2: Create Questions
Go to the Question Management section and add quiz questions.

Step 3: Generate Quiz
Select the desired questions and generate a quiz.

Step 4: Take the Quiz
Students can answer the generated quiz using the system.

Step 5: Check Results
View quiz scores and results after submission.

**How to Run the System**

1.	Download or clone the project repository.
2.	Open XAMPP and start Apache and MySQL.
3.	Place the project folder inside the htdocs directory.
4.	Import the database file into phpMyAdmin.
5.	Open the browser and type:
http://localhost/quiz-generator
